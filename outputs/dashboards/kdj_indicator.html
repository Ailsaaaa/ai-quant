📖 KDJ 是什么？

KDJ 由中国投资者在 Stochastic（随机指标）基础上改良加入 J 线而成，与 MACD、RSI 并称A股"三大经典指标"。它的核心思想是：通过比较收盘价在近N日高低区间的相对位置，判断短期买卖力道。

计算公式

RSV = (C−L₉)/(H₉−L₉)×100
K = ⅔×前K + ⅓×RSV
D = ⅔×前D + ⅓×K
J = 3K−2D

判断规则

信号条件大白话
🟡超买J>100 或 K>80"涨太猛了，该歇歇了"
🟢超卖J<0 或 K<20"跌过头了，可能要反弹"
📈金叉K 从下往上穿过 D"短期可以买了"
📉死叉K 从上往下穿过 D"短期该卖了"

KDJ vs 其他指标

指标灵敏程度适合
MACD较慢中长期趋势
RSI中等超买超卖确认
布林带中等震荡市高低点
KDJ最快短线交易信号、顶底背离
图1 · 长江电力收盘价走势
📌 大白话解读：先看价格走势有个整体印象。长江电力近3年像一条缓缓向上的坡道，虽然有波动但总体趋势向上。最高点和最低点已用箭头标出。
图2 · KDJ 三线完整视图
📌 大白话解读：粉色K线、蓝色D线、紫色J线。三条线高于80是超买区，低于20是超卖区。紫色J线最灵敏，它是"快线中的快线"——J线超过100或跌破0都是值得关注的极端信号。当前J=91.4。
图3 · K/D 交叉信号
📌 大白话解读：只看K和D两条线。粉线从下往上穿过蓝线就是金叉（买入信号），从上往下穿过就是死叉（卖出信号）。注意KDJ的金叉死叉比较灵敏，单独使用时会有不少假信号，最好结合其他指标一起看。
图4 · J线 vs RSI
📌 大白话解读：紫色实线是J线，橙色虚线是RSI。可以看到J线的波动幅度明显大于RSI——这就是KDJ"灵敏"的代价：给出的信号更多，但也更容易出假信号。两个指标一起看能提高判断的准确性。
图5 · 三股 J 线大对比 — 谁最"激动"？
📌 大白话解读：把三只股票的J线放在一起一目了然：红色中芯国际的J线上下翻飞最剧烈，典型的高波动股；绿色长江电力最平稳，J线很少进入极端区域；橙色比亚迪介于两者之间。可以说：J线的波动幅度直接反映了一只股票的"脾气"。
